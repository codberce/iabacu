# AGENTS.md

- Aim for a calm, modern, open-source educational tool: useful and trustworthy, never flashy.
- Keep the experience Bacalaureat-first while making olympiads easy to find.

- Favor clear hierarchy, generous spacing, restrained colors, consistent components, and subtle interactions.
- Make information easy to scan on every screen size; preserve useful context and accessibility.
- For GitHub, use a descriptive unprefixed branch, sync with `master`, validate the work, and open a concise PR.
